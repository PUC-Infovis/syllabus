Idioms para jerarquía y d3-hierarchy

Node-link diagrams

Diagrama de topologia que utiliza marcas discretas para nodos y enlaces.

Tree

Sigue el orden jerárquico y mantiene cada nivel de profundidad alineado.

Cluster (Dendrogram)

Sigue el orden jerárquico pero alinea los nodos más profundos (hojas).

Adjacency diagrams

Diagrama que representa topología usando posición relativa entre nodos.

Partition

Puede utilizarse un rectangulo como marca y usar área como encoding para algún atributo.

Enclosure diagrams

Diagrama que representa topología usando contención entre elementos y utiliza el área como encoding para algún atributo.

Treemap

Subdivide el espacio recursivamente en rectangulos. Es un enclosure diagram eficiente en espacio.

Pack

Subdivide el espacio recursivamente en paquetes circulares. Menos eficiente en espacio pero comunica más efectivamente la contención.
